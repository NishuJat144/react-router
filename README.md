# React Router 🚀

A complete React Router learning repository that covers routing concepts from beginner to advanced using **React Router DOM**. This repository contains hands-on examples to understand modern client-side routing in React applications.

---

## 📚 Topics Covered

- Introduction to React Router
- BrowserRouter
- Routes & Route
- Link & NavLink
- useNavigate
- Nested Routes
- Dynamic Routes
- Route Parameters
- useParams
- Search Parameters
- useSearchParams
- Layout Routes
- Outlet
- Index Routes
- Error Pages
- useLocation
- Protected Routes
- Loaders
- Actions
- Forms in React Router
- Data Fetching
- Navigation State
- Error Boundaries
- Lazy Loading Routes
- Code Splitting
- 404 Not Found Page

---

## 🛠️ Tech Stack

- React
- React Router DOM
- JavaScript (ES6+)
- CSS
- Vite

---

## 🎯 Purpose

This repository is created for learning and practicing React Router concepts through practical examples. It also serves as a quick revision guide for interviews and real-world React development.

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/react-router.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

After completing this repository, you will be able to:
